# Netflix Clone 🎬

A fully responsive **Netflix UI Clone** created as part of my web development journey. This project focuses on recreating the look and feel of the popular streaming platform using modern web technologies.

## 🚀 Features
* **Landing Page:** Professional-looking hero section with background images.
* **Responsive Design:** Optimized for mobile, tablet, and desktop screens.
* **Modern UI:** Used clean CSS and semantic HTML.
* **Interactivity:** Includes sections like movie grids and a sign-up area.

## 🛠️ Tech Stack
* **HTML5:** For structure.
* **CSS3:** For styling and layouts (Flexbox/Grid).
* **JavaScript:** For basic interaction (if applicable).

## 📂 How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
